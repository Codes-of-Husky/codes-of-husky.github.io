周刊（2019 第 19 周）

每周新鲜事📅


过去一周的有趣见闻。

#新闻

> Linux 5.1 发布

本周 05 月 05 号，Linus Torvalds 在内核邮件列表上宣布释出 Linux 5.1，5.2 合并窗口开启。

> 微软在开发者大会上宣布 Windows Terminal

在本周 05 月 07 号于美国华盛顿州雷德蒙德举行的微软开发者大会上，微软宣布推出新的命令行终端，将支持主题、标签和绘文字，在功能它将能媲美 Unix 终端。Windows Terminal 将通过 Microsoft Store 在今年六月提供给用户。微软称它倾听了用户的呼声，新的终端支持多个标签；使用 GPU 加速的 DirectWrite/DirectX 文本渲染引擎，用户可选择使用新的字体；提供多种设置和配置选项，让用户更好的控制终端环境。并且，最重要的是，它将开源。另继去年 12 月，微软正式确认会把 Edge 改以 Chromium 架构来迎合市场需求外，微软也在开发者大会上提到了新的 Edge 浏览器将加入 Internet Explorer 模式，目的是为了让使用者更简易对应迟迟未肯升级的旧式网页。

> Red Hat Enterprise Linux 8.0 正式发布

Red Hat Enterprise Linux（常缩写作 RHEL）是一个由 Red Hat （红帽）公司开发的面向商业市场的发行版，以出色的稳定性著称。其基于 Fedora 构建。该项目于日前发布了 Red Hat Enterprise Linux 8.0 版，后者基于 Fedora 28 及 Linux 4.18 内核。 此前的大版本更新 RHEL 7.0 发布于 2014 年，基于 Fedora 19 及 Linux 3.10 内核。因此，可以预见这一新的大版本将包含巨量的更新。

> GitHub 推出 GitHub Package Registry

这是一个软件包管理服务，可以轻松地基于源代码发布公共或私有软件包，开源项目可永远免费使用，将软件包存储在与源代码相同的安全环境中，所有这些都受 GitHub 证书保护。

#NBA

> 休斯顿-火箭止步季后赛

经过了一周的激烈角逐，金州-勇士队以 4：2 战胜休斯顿-火箭队，晋级西部决赛。密尔沃基-雄鹿队以 4：1 战胜波士顿-凯尔特人，晋级东部决赛。

西部决赛的候选队将在波特兰-开拓者队和丹佛-掘金队中出现，目前两队总比分 3：3 平，将在接下来的抢七大战中一决胜负。同样，在东部，费城-76人队和多伦多-猛龙队也是 3：3 战平，抢七大战将在下周一 05 月 13 日举行。

#资源

> Calculator

微软开源了 Windows 内置的 Calculator，这是 GitHub 上的 Calculator 项目。此项目实现语言为 C++ 代码并不复杂，快来看看微软工程师编写的代码吧！

> about 协议的来历（英文）

“about: “开头的网址，用来返回与浏览器本身相关的内容，最常用就是空网址 about: blank 以及 about:history，本文讲述了这个协议的来历。

> 数据挖掘教材（英文）

《数据挖掘》（第二版）这本书本身没有全部开源，这个网页提供了所有章节的 PPT 教辅材料和实验代码。

> 25 种团队通信工具（英文）

Slack 是目前最流行的团队协同通信工具。这篇文章列出了25种 Slack 的替代品。
