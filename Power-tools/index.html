Power Tools

工欲善其事，必先利其器🛠

使用好的工具可以达到事半功倍的效果，本文为你介绍一些运行在 Microsoft Windows 平台下的优秀工具软件。

个人工具集

安全工具

名称 介绍
Avast 来自捷克，全球十大软件之一。拥有免费版本，免费版的功能足够个人用户使用
Norton 赛门铁克公司出品的杀毒软件，全球十大杀毒软件之一
Kaspersky 来自俄罗斯的杀毒软件，全球十大杀毒软件之一
火绒安全 国产主动防御工具，自定义防御功能强大，但是杀毒功能较弱
GlassWire 防火墙工具。以图形的方式显示网络流量，可以针对每一个联网程序进行防火墙控制
Firefox Monitor 谋智火狐提供的安全工具，可以用来监测你的邮箱是否遭到了泄露

任务管理工具

名称 介绍
Microsoft To-Do 微软出品，奇妙清单的替代者
Trello ATLASSIAN 出品，一款不错的团队协作工具，也可以用来管理个人事项。我还用来在电脑和手机以及 iPad 之间同步文字和图片信息。

清理工具

名称 介绍
CCleaner 由 Piriform 开发的一款软件，可从 Windows 及其他软件中清理不必要的垃圾文件

远程连接工具

名称 介绍
PuTTY 一款开源的 SSH 和 Telnet 工具
TeamViewer 远程桌面工具
XSHELL NetSarang 出品，远程控制终端
XFTP 和 XSHELL 一家公司的产品，FTP 工具

影音文件查看工具

名称 介绍
完美解码 国产影音播放器，专注播放功能，功能强大
Honeyview 来自 Bandisoft 的图片查看工具，功能强大
foobar2000 轻量可扩展的音乐播放工具
Seer 功能类似 Quicklook
Quicklook 将 macOS 上的空格预览工具带到 Windows 上
Sumatra PDF 小巧快速的PDF查看工具
ABBYY FineReader OCR 文字识别软件

录屏截图工具

名称 介绍
ScreenToGif 录屏工具，可将屏幕内容变成 Gif
Snipaste 差不多是功能最强大的截图工具了
Camtasia 屏幕录制和视频剪辑工具
Snagit 截屏工具

下载工具

名称 介绍
迅雷 下载工具，但是官方推出的产品并没有只专注于下载，因此只建议使用极速版或者阉割过的官方版本
uTorrent 使用 p2p 协议的下载工具

文件整理工具

名称 介绍
WinMerge 可以用来比较文件或者文件夹差异的小工具
Meld 可以用来比较文件，也可以当作 mergetool
Kdiff3 可以用来比较文件，也可以当作 mergetool
Speecy 由 Piriform 开发的一个软件硬件信息工具，可以查看电脑内的各类硬件信息以及 CPU、显卡、硬盘温度等
Defraggler 由 Piriform 开发的一个磁盘碎片整理工具
Recuva 由 Piriform 开发的一个可以找回已删除文件的工具
SpaceSniffer 用来查看电脑磁盘的占用情况
FreeFileSync 开源的文件比对和备份工具
FastCopy 快速的复制备份文件

搜索工具

名称 介绍
Everything 超级强大快速的本地文件搜索功能
Listary 强大的文件管理和搜索功能

解除占用

名称 介绍
LockHunter U 盘不能弹出？文件不能移动、删除？它帮你找到元凶
USB Safely Remove 安全的移除 U 盘硬盘

效率提高工具

名称 介绍
Ditto 免费开源的剪贴板增强软件，功能强大
Pomotodo 番茄工作法软件
FocusBooster 番茄工作法计时器
Visual Studio Code 微软的开源文本编辑器，本站就是使用它建立的，本文也是使用它写的
iSlide 基于 PowerPoint 的插件工具，助你快速做出美观大气的 PPT
git 本博客中有一篇博文对此工具做了专门介绍
欧陆词典 多语种词典，可自定义字典
HistCite Pro 文献引文分析利器
Bandzip for Windows 一款来自韩国的开发商 Bandisoft 开发的 Windows 平台压缩软件。界面简洁，功能强大
Sysinternals 微软的官方工具套装，用来管理诊断 Windows 系统或者应用程序的问题
Sigcheckgui 微软 Sysinternals 工具中 Sigcheck 的可视化版本，可以用来批量检查文件的数字签名。它能够验证文件的完整性、保证不被篡改，标明文件的发布日期，还能减少杀毒软件的误报

工具网站

名称 介绍
小众软件 小众软件是一个分享免费、小巧、有趣、实用软件的网站
MSDN，我告诉你 可以在这里下微软的所有产品
善用佳软 优秀软件推荐
书伴 一个关于 Kindle 设备以及 Kindle 电纸书的网站。在这里你可以找到精选书籍以及 Kindle 相关的软件
廖雪峰的官方网站 技术分享型网站，由廖雪峰建立并维护，提供：编程知识、JavaSE 教程、数字货币教程、JavaScript 教程、Python 教程、SQL 教程和 Git 教程，全是干货
中华诗词 廖雪峰建立并维护的网站，中华诗词收录了上至先秦，下至当代共计六万余首诗词，完全免费并开放所有诗词内容和程序源代码。每天网站更是会推荐一首诗词放在主。
马克思主义文库 旨在传播马克思主义和共产主义历史文献和参考资料的一个网站
中國哲學書電子化計劃 中國哲學書電子化計劃是一個線上開放電子圖書館，為中外學者提供中國歷代傳世文獻，力圖超越印刷媒體限制，通過電子科技探索新方式與古代文獻進行溝通。收藏的文本已超過三萬部著作，並有五十億字之多，故為歷代中文文獻資料庫最大者
