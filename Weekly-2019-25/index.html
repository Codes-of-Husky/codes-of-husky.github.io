过去一周的有趣见闻。

#新闻

> 重新设计的 Windows Terminal 释出 Windows 10 预览版本

微软如约带来重新设计的 Windows Terminal，上架到 Microsoft Store 的虽然仍是预览版本，但他们承诺会有大多新功能都已经能让一众前阶使用者尝鲜了，包括可以在一个工具中使用 Command Prompt、PowerShell 和 Linux 语言，支持标签分页、自定义、网页链接等。

> 苹果召回部分存在电池发热隐患的 MacBookPro

苹果技术网站发布公告，宣布了 15 英寸 MacBook Pro 电池召回计划。苹果称，它确定少数较早推出的 15 英寸 MacBook Pro 电脑中的电池可能会过热并造成消防安全隐患。
